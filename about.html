RemoteStick is all about this. Remotely controlling your home.

Tellstick

RemoteStick revolves around Tellstick. Tellstick, a USB stick manufactured by the Swedish company Telldus, "turns any computer into a wireless home automation center by being the link between your computer and electronics & lights" (quoted from Telldus web site). The Tellstick can be bought from droidstuff.se or see Telldus list of retailers. Tellstick is able to control all kinds of devices from different manufacturers such as power sockets, light bulbs, switch converters and many more.

How does it work?

You simply plug the Tellstick USB stick to a computer at home. It need to run either Windows, Mac OS X or Linux with Python installed (preferably version 2.7). The Tellstick driver is installed by downloading the appropriate package from Telldus homepage. Now RemoteStick Server is installed which, when started, will communicate with the Tellstick through the driver and expose it on the network using RESTful services. Now you will be able to access your server using the Android client on your phone.
